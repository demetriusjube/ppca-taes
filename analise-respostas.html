Análise de resultados do IMI - Feedback da sessão de julgamento¶

Este notebook procura fazer a anlise das respostas da pesquisa de feedback da sesso de julgamento do TST. Para isso, as seguintes aes sero realizadas:

  • Recuperao das perguntas formuladas, para mapeamento das questes conforme o id do IMI
  • Identificao de quais perguntas so reversas (Tp = R)
  • Mapeamento das respostas para cada pergunta
  • Clculo do valor da resposta das perguntas reversas seguindo a frmula: (Valor mximo Escala de Likert + 1 ) - Valor da resposta
  • Clculo da confiabilidade com o Alpha de Cronbach
  • Clculo da Mdia e Desvio Padro
  • Similaridade: Correlao de Pearson

Importação das bibliotecas¶

Utilizaremos as seguintes bibliotecas:

  • pandas: Permite criar DataFrames a partir dos arquivos Excel
  • numpy: Biblioteca estattica do Python
  • pingouin: Lib que permite o clculo do Alpha de Cronbach
  • matplotlib e seaborn: Bibliotecas que permitem plotar grficos

def getQuantidadeDeTecnicas(media:float)->int:
    if (media >= 4.5) :

    if (media < 4.5 and media >= 3.5):
        return 3
    if (media <3.5 and media >=2.5):
        return 2
    if (media <2.5 and media >=1.5):
        return 1
    return 0
